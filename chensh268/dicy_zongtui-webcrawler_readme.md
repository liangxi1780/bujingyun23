#zongtui-webcrawler

众推，基于机器学习和深度学习的开源项目，主要处理网络爬虫、文本挖掘等相关的内容！

1）实现基于hadoop思维的分布式网络爬虫，在完善和深入。
2）实现各种包括去重分析、关键字提取、情感分析、文本分类处理等。

核心设计思路是可以接入任何一种爬虫，并且可以针对此种爬虫的问题进行扩展。
