## 部署说明

 - 运行环境 windows/liux

 - windows 见 IIS 托管 .Net Core 2.1 帮助文档

 - Liux 见 帮助文档

 - 数据库初始化 见 "Core.DataAccess.Model" 采用 ef code first 初始化 配置帮助 脚本 "note.cs" 根据说明 执行指令 可自动生成 数据库关系表

 - 需要 Redis服务

 - 支持集群部署 appsettings.json 文件配置

```

{
"Service": "A_Service",
"QueueTimedLoop": {
"length": 1, // 实例数量
"parameter": {
"threadLength": 10, // 每实例任务处理线程数量
"sleep": 200 // 处理任务后休眠时长
}
}

};

```
