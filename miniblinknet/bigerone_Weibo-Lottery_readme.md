# 微博抽奖助手

#### 项目介绍

1. 自动根据关键词搜索热门抽奖微博，根据微博内容自动转发、点赞、关注要求博主。
2. 模拟人为行为，自动上网浏览微博，进行随机的点赞、转发、发布微博操作，快速培养微博新号，防止被识别为机器人。
3. 灵活自如的操作间隔设置，防止操作过于频繁被关进微博小黑屋。
