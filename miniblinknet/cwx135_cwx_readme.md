#### 介绍
磁盘坏道检测。

#### 软件架构
软件JAVA编写的。

#### 使用说明

1.  按复制键填充文件，完成后U盘最好重新插入。
2.  再按检测键，完成后就知道是否有坏区。如果有坏区在磁盘上TEST目录下就有坏区的编号文件。
