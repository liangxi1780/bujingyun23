#记录说明
* com.abstr  (记录关于抽象类的一些继承)

* com.adapter (适配器模式简单应用)

* com.gc  (对GC机制的一些学习)

* com.jms (JMS通信的一些记录,如activeMQ等)

* com.method (运行时常量池的导致的内存溢出)
