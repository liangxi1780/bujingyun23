# WanAndroid
采用Android Architecture Components (LifeCycle, LiveData, ViewModel, Room) MVVM 架构，
结合 DataBinding + Retrofit + RxJava 使用玩Android开放API，通过组件化模式构建WanAndroid 客户端。
