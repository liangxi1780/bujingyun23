# 多端异构的数据同步工具

- 借鉴了Kettle的Step+Transform概念，高吞吐量多线程本地数据管道DataSet设计

- 借鉴了Flink的PipeLine思想概念

- 使用Scala编写

- 配合数据预处理平台管理端WEB后台，支持可视化配置，监控，调度和管理

- 支持指标输出，可通过Grafana展示
