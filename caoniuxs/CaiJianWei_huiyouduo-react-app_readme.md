# huiyouduo-react-app

#### 项目介绍
惠又多

#### 软件架构
react全家桶（axios，react-redux，react-router），webpack打包工具，hbuilder打包成webapp

#### 安装教程

1. git clone https://gitee.com/CaiJianWei/huiyouduo-react-app.git

#### 使用说明

1. yarn install 安装依赖
2. yarn dll  常用的插件封装成dll（已经安装打包提交）

3. yarn dev 运行
4. yarn build 打包
