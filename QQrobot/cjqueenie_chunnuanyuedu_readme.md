# 春暖阅读

#### 介绍
一款基于uniapp的阅读器

#### 软件架构
基于Uniapp的H5项目


#### 安装教程

1. git clone地址
2.  使用HBuilderd把整个项目文件导入
3.  HBuilder启动编译即可
4.  特别说明，需要去Node文件夹下 ，开启测试接口服务
