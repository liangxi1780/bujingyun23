#MovieMgr
本项目使用JFinal + h-ui开发。
是要给附近的一个小作坊5D电影院弄个查询系统。
不提供用户管理， 只有影片管理功能。

#简单介绍
sql目录中的*.sql文件为建表语句。
工程为标准的eclipse工程。
