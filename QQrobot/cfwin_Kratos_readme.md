# Kratos

A clean, simple and responsive blog theme of WordPress, based on [Bootstrap](http://u.720life.cn/g/54145d0471d91890860f7f8463c030465a8160a390414e5887ea7b27f65f8894df8a88b45f3cfc9c4098552eeb42db05) and [Font Awesome](http://u.720life.cn/g/54145d0471d91890860f7f8463c030461b497c4d42fd57c4cbd279a2424a9c471d20fd341f46c05db63782dc849dfeca). Created and maintained by Vtrois. Stay up to date with the latest release and announcements on [Bulletin Board](http://u.720life.cn/g/54145d0471d91890860f7f8463c03046b36a21e4ef5b31ff6564064851fa19220d6adc6ae8c422d15a0a3b6a6393cae1).
