##分布式缓存中间件##
  方便实现缓存的分布式，集群，负载均衡，故障自动转移，并兼容多种缓存存储的分布式缓存中间件。 用于解决分布式架构中的分布式缓存环节。

##特点：##
 1. 代码少，便于扩展。
 2. 兼容阿里云memcache，redis，ssdb。
 3. 规范缓存使用接口，屏蔽底层缓存实现。
 4. 通过配置连接字符串即可切换不同存储引擎，可以混合不同存储引擎组成缓存集群部署。（如部分redis，部分memcache）
 5. 动态负载均衡，故障转移，线上无缝平行扩展和扩容，方便运维。
