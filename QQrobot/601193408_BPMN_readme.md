## 介绍 

找一个好用的画图工具真心不容易，Activiti 工作流自带的 Web 版画图工具，外表挺华丽，其实使用起来各种拧巴；Eclipse 的 Activiti 画图插件，对于相对复杂的流程也是很不友好。

## 环境搭建

| ---- | ----- |----- |----- |

## 功能模块

这是一个Python版本，Java版本功能已经基本开发完毕，需要进行功能迁移。

- 用户登录
- 流程列表(CURD)
- 用户注册(待实现)
- 游客访问在线作图，可实现在线导入导出，本地缓存
