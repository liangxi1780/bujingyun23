**特色：**
1. 代码量极少，可读性强。核心代码仅1千行左右。
2. 学习门槛低，二次开发只需实现两个接口。
3. 资源利用率高，性能爆表，充分压榨CPU、带宽。
4. 提供丰富的插件式服务，包括：心跳插件、断链重连插件、服务监控插件、黑名单插件、内存池监测插件。
