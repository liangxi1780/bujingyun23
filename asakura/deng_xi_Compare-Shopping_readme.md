# Compare  Shopping

#### 介绍

##### 基于JavaWeb的比价商城

#### 软件架构
```
菜鸟学习项目:
    基于servlet开发的比价商城项目
    分为商城前台和后台管理模块
    并未实现购物功能，一个纯粹的购物比价平台
    适合新手参考学习
```

#### 安装教程

```
1.  本项目jdk版本采用1.8，tomcat8.5.11版本
2.  将项目克隆下来，将doc文件夹下的sql文件导入本地数据并修改数据库配置信息
3.  运行截图
```
