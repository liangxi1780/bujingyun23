# 趣享GIF开源版 v1.0.3

 

趣享GIF是一款Android端开源的GIF浏览和分享App，界面基于Material Design标准进行设计。App围绕GIF为主题，建立了一个小型的社交系统，在这里你可以：

 * 查看热门搞笑的GIF图
 * 关注你喜欢的人，他的有趣分享尽收眼底
 * 有好玩的内容想让大家知道？一键发布你自己的GIF趣图
 * 对你感兴趣的内容点赞、点评，和众多网友交流
 * 喜欢的内容轻松转发至主流社交软件，传递你的快乐

更多好玩的东西等待你来探索，赶快开启你的快乐之旅吧。
