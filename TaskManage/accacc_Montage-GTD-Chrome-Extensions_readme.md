# Montage GTD Chrome Extensions

Montage GTD的谷歌Chrome浏览器扩展，支持右键分享选中文字、链接、图片，支持屏蔽娱乐网站开启高效工作模式，支持番茄工作法待办事项

## 支持
- 番茄钟提醒
- 右键分享选中文字、图片、链接、当前浏览页

## 即将支持
- 高效工作模式
- 番茄工作法
- 待办事项
- 登录/注册
