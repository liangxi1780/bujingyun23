Blog2
=======

这个Blog项目基于Yii2.0预览版，并且随Yii2.0项目进度进行更新

## 部署

1. Web服务器的文档根目录指向 www/
2. 修改 www/index.php 中的下面这行 
```
require 'yii2/framework/yii.php';
```
 为框架的绝对或www相对的路径
