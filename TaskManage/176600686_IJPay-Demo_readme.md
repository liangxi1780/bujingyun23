- IJPay是[JPay](http://u.720life.cn/g/54145d0471d91890860f7f8463c0304651d2306d5c928660701d2639705351d9)后台接口SDK不依赖任何第三方 mvc 框架，仅仅作为工具使用简单快速完成支付模块的开发，可轻松嵌入到任何系统里。
- [JPay](http://u.720life.cn/g/54145d0471d91890860f7f8463c0304651d2306d5c928660701d2639705351d9) 是简易而不简单的Android 支付SDK，JPay是对微信App支付、支付宝App支付的二次封装,对外提供一个相对简单的接口以及支付结果的回调。
- IOS支付SDK，敬请关注。
