# task_frontend

#### 介绍
任务管理系统的对应前端项目
主要为了在项目中实践和应用TypeScript和Nuxt服务器端渲染技术

#### 软件架构
vue
vue-router
vuex
Nuxt
elementUI
TypeScript


#### 安装教程

1. git clone 仓库地址
2. cd 项目目录
3. npm i
4. npm run serve

#### 使用说明

1. cd 项目目录
2. npm run serve

#### 参与贡献

1. 慕课网 Node.js 从零开发 web server博客项目 前端晋升全栈工程师必备
