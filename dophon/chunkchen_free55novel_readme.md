### 功能：###

1. 小说栏目管理

2. 小说管理

3. 小说章节管理

4. 用户管理

5. 管理员管理

6. 用户可进行收藏、推荐

7. 按年月日独立统计的排行系统


### 特色：###

1. 基于LAMP上 Yii 框架，扩展灵活，开放快速

2. 基于Bootstrap 前端框架，支持手机端等移动设备访问

3. 开源、免费 方便快速二次开发
