 

# goose 使用详情

1. 依赖使用 go mod

   > go mod 使用事项
   >
   > 1. 禁用go path
   > 2. goland 中 setting -> go -> go module(vgo) -> enable go modules (vgo) intergration 打钩 即可
   > 3. 缺少依赖的时候 在项目文件夹中 cmd -> go get ./...
   > 4. 查看依赖下载在哪里可以用 go list -m -json all
   > 5. 其他请百度
