# General.CMS
通用企业官网基础框架

### 环境框架
windows 10、vs2017、支持多种数据库、Asp.Net Core MVC 2.2

### 后端
* Repository + Service
* Async和Await 异步编程
* Cors 简单的跨域解决方案
* AOP切面日志
* 基于[Banana.Uow](http://u.720life.cn/g/54145d0471d91890860f7f8463c030460a852bece5ad5ab5ab502f3cd5b47d0edd774c066e27e129a6a6f3afd184aab0) ORM，支持多数据库
* Authorize 基于角色授权
* 其他
