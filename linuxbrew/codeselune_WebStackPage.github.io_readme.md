怎么用?
---
你可以像我一样直接使用静态托管形式，如果你需要一个后台系统方便管理，可以参考下面的第二第三种解决方案：
#### 方法1. 使用静态托管
最简单快速上线自己的导航网站，你可以直接**下载**本项目修改内容既可部署上线。
