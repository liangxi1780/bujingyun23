# random

#### 介绍
random 返回两个浮点数之间的随机浮点数

#### 安装教程

1. composer require DaHne/Random

#### 使用说明
````
$i = 0; 
while ($i < 100) {
    $s = random_int(-100, 20);
    $e = random_int(-20, 100);
    $sf = random_int(10000,99999);
    $ef = random_int(20000,99999);
    echo '$s ' . $s.'.'.$sf . ' $e ' . $e.'.'.$ef . ' $res ' . $r($s.'.'.$sf, $e.'.'.$ef);
    echo PHP_EOL;
    $i++;
}
````
