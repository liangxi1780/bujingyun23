# Rental Listing Inquiries
Rental Listing Inquiries 数据集是 Kaggle 平台上的一个分类竞赛任务，需要根据公寓的特征来预测其受欢迎程度（用户感兴趣程度分为高、中、低三类）。其中房屋的特征 x 共有 14 维，响应值 y 为用户对该公寓的感兴趣程度。 评价标准为 logloss。
