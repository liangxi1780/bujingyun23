#CameraCollect
该项目采用QT编写的一个小工具。
主要为解决opencv中训练图像特征的过程中，采集图片比较麻烦的问题。该项目采用连接电脑的USB摄像头，提供多种模式采集图像，并生成opencv_traincascade.exe的训练环境与脚本。
不断完善中
