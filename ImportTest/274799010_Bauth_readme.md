Bauth权限系统
===============

Bauth权限系统基于Thinkphp5开发的免费开源权限系统，是一个开源免费的权限系统

Bauth权限系统的目标是做一个权限完善且多模块的后台管理系统，为开发项目减少开发后台的时间
系统使用了composer autoload和 namespace 的特性，重要模块独立化，方便系统的扩展与开发

 + 独立的Menu模块
 + 独立的Bauth权限模块
 + 多模块，多角色的权限验证
 + 后台引入优秀的Jqeury插件，方便快速开发后台
