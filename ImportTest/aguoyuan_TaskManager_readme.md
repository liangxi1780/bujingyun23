#TaskManager
任务信息提示工具。特定功能的定制软件。逻辑可参考，源于TaskTip，功能外部无用。

实现功能:
包含自定义控件的定义与使用，自定义控件属性与响应事件、
包含Json数据的解析与转化、
包含无边框窗体的移动、尺寸调整
包含应用程序自动更新逻辑、
包含应用程序单例启动逻辑
