#欢迎使用 iCMS 内容管理系统

iCMS 是一套采用 PHP 和 MySQL 构建的高效简洁的内容管理系统,为您的网站提供一个完美的开源解决方案

##iCMS 可以用来干什么
- 博客
- markdown博客
- 资讯/新闻站点
- 公司产品展示
- 图片展示
- 快速二次开发
- 移动端网站
- 接入微信公众平台
- 为微信小程序提供API服务

##环境需求
- [PHP 5.3](http://u.720life.cn/g/74600cc848fc21935f3dab9fc998167e886392f56f4cbe6ed24a77ecd7b8c3e5) 以上版本 **推荐使用PHP5.6以上版本**
- [Mysql5](http://u.720life.cn/g/b82b5139a48b1e95aee5194da6c4350a106fdf487f5a8e8f7f7ee2dd2f0304a7) 数据库支持，并在PHP中安装了相关扩展
- [PHP cURL 扩展](http://u.720life.cn/g/d72b6b65beb51348ff2c110f8fad4b1297838c564d8680c843eb5ad7af22d8d48ac3d316dfb1f0cc10296b5ed5d9165f)
- [PHP mbstring 扩展](http://u.720life.cn/g/d72b6b65beb51348ff2c110f8fad4b12a8d45d308a9c52150fecc7d9f8654a23eb160b74ca63a87ab221e18453620f90) 或者 [PHP_iconv 扩展](http://u.720life.cn/g/d72b6b65beb51348ff2c110f8fad4b124d553e87a393334b5c53223d350cb18323e3b379d5e3f7c747d6e7b20e41c5e1) 支持
- [PHP_GD 扩展](http://u.720life.cn/g/d72b6b65beb51348ff2c110f8fad4b12fe0aeee9fa4bc5b479317d4c4f6cf046d917564c1acb82bbeb33d182cee16f47) 验证码程序需要用到
- Redis 或者 Memcache 缓存 [可选]
