# 智能仓储管理系统-group3
# 1.项目简介
	本项目是基于A9主控板 M0远程板 实现的智能仓储物联系统。M0板负责远程数据的监控，并将监控的数据发送至A9主控板。A9板本身搭载了BOA服务器，当A9板接收到数据便将数据发送给服务器。用户可以通过网页登录，可以查看到仓库的信息，并且可以设置M0板的LED,蜂鸣器等的状态。
