### 她是什么，就是一个cms，一颗小小的小糖果 
1. 她简单。没有多站点，没有采集，没有多语言。真的不是我们偷懒，这些功能真的很少用到  
2. 她青春。都是新的技术，新的版本。新技术代表生产力，新不代表不稳定，您可以保留意见
3. 她活跃。我们会尽量经常更新，尽量......
4. 她年轻。她还没满周岁，有很多毛病，请大家谅解
