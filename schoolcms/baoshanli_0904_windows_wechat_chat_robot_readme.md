# windows 微信聊天机器人

#### 项目介绍
PY写的 支持微信 个人聊天，微信群聊，
360报毒，信任就可以了。



#### 使用说明

config.json 文件里面可以控制开启 自动聊天 是针对 “个人还是 群聊”
{
  "wx_solo":1, //个人聊天
  "wx_group":0// 微信群聊天
}

双击wechat_bot.exe 等待一会，扫描出现的二维码，登录微信账号， 然后 别人和你对话的时候  就会自动回复了。

注： 不能登录网页版微信的微信账号， 也不可以使用此软件
